# Slide 17  
**第1章 狭义相对论的引入**  

由“时间的绝对性（$t' = t$）”，可推出“同时性的绝对性”，其含义如下。设 $p_1, p_2$ 是两个事件，在惯性系 $K$ 的时空坐标分别为  
$$p_1 = (t_1, x_1, y_1, z_1), \quad p_2 = (t_2, x_2, y_2, z_2).$$  
若 $t_1 = t_2$，则称 $p_1$ 和 $p_2$ 对 $K$ 系为同时事件。再设 $K'$ 系赋予 $p_1, p_2$ 的时空坐标为  
$$p_1 = (t_1', x_1', y_1', z_1'), \quad p_2 = (t_2', x_2', y_2', z_2'),$$  
由“时间的绝对性”[式 $(1\text{-}1\text{-}4)$ 的第一式] 得 $t_1' = t_1$, $t_2' = t_2$，故 $t_1 = t_2$ 蕴含 $t_1' = t_2'$。这表明对 $K$ 系同时的两事件对 $K'$ 系也必然同时，即“同时性的绝对性”。牛顿力学默认伽利略变换，故在其中同时性是绝对的，且符合日常经验。然而爱因斯坦经反复思考后质疑此结论的普适性，并构想如下反例：  

一列火车 $A'B'$ 匀速飞驰于地面。假定地面系观测到车头 $A'$ 和车尾 $B'$ 同时遭受雷击（图 1-2）。爱因斯坦指出，对火车系而言，这两个事件**并非同时**，理由如下：  

> [!NOTE] 🖼️ Figure 描述  
> 图 1-2 展示爱因斯坦列车的雷击同时性问题：  
> - **水平布局**：两条平行水平线，下方为地面系（静止），上方为火车系（运动）；  
> - **地面系标记**：从左至右依次为 $B$（左端点）、$M$（中点）、$A$（右端点），$M$ 为 $AB$ 段中点；  
> - **火车系标记**：从左至右依次为 $B'$（车尾）、$M'$（中点）、$A'$（车头），$B'$ 与 $B$ 垂直对齐，$A'$ 与 $A$ 垂直对齐；  
> - **事件标注**：$B'$ 和 $A'$ 位置各有一个闪电符号，表示雷击事件；  
> - **运动方向**：火车系整体向右匀速运动，右端有水平向右实线箭头标注 $v$；  
> - **关键细节**：所有点标记使用正体（$A,B,M$ 为地面系，$A',B',M'$ 为火车系），闪电符号置于 $B'$ 和 $A'$ 正上方，速度矢量 $v$ 明确标注于箭头末端，便于 TikZ 重绘时保留坐标对齐与运动可视化。  

以 $A$ 和 $B$ 分别代表地面系中雷击发生时 $A'$ 和 $B'$ 在地面的印记（忽略火车高度，故 $A' \equiv A$, $B' \equiv B$）。所谓“地面系认为雷击同时发生”，是指车头与车尾发出的光**同时传至中点 $M$**<sup>①</sup>。设 $M'$ 为火车中点，因火车向右运动，$M'$ 迎向车头来光而背离车尾来光，故先收到车头来光、后收到车尾来光。由此，火车系判定两雷击事件**不同时**。爱因斯坦由此出人意料地质疑“同时性的绝对性”，实质是否定伽利略变换式 $(1\text{-}1\text{-}4)$ 中的 $t' = t$，并以此为突破口创立狭义相对论。其奠基性论文《论运动物体的电动力学》发表于 1905 年，首次提出狭义相对论的两个基本假设：  

**假设 1（相对性原理）** 所有惯性系平权，任何物理定律在所有惯性系中的数学表达式均相同。  
**假设 2（光速不变原理）** 真空中的光速沿任何方向、对任何惯性系均为 $c$，且与光源运动无关。

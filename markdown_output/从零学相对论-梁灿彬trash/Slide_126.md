### §6.2 弯曲时空的物理定律（续）与 §6.3 等效原理导言  

正如 Slide 125 所强调，**"坐标光速" 的表述具有坐标依赖性**：当选用的坐标系恰当时，其值可任意变化（既可 $ \neq c $，也可 $ > c $），因此物理意义薄弱。"光子速率" 与 "质点速率" 的本质问题应通过 **4 维语言** 描述，即聚焦于 **世界线几何性质**：  
> 光子和质点的运动由其世界线的几何特征决定，而非坐标速率。  

请回顾 §3.3 中关于极限速率的讨论（狭义相对论基本信条）：  

#### **3 维 vs 4 维语言表述对比**  
| 语言类型 | 光子表述                                      | 质点表述                                      |  
|----------|---------------------------------------------|---------------------------------------------|  
| **3 维** | (A) 相对于任何惯性系速率 $ u = c $（或 $ u = 1 $） | (B) 相对于任何惯性系速率 $ u < c $（或 $ u < 1 $） |  
| **4 维** | (a) 世界线为 **(闵氏时空) 类光曲线**（$ ds^2 = 0 $） | (b) 世界线为 **(闵氏时空) 类时曲线**（$ ds^2 > 0 $） |  

**关键推广逻辑**：  
- 3 维表述依赖 **(整体) 惯性系**，而弯曲时空中 **不存在整体惯性系**，故 (A)、(B) 无法直接推广；  
- 4 维表述可自然推广：将括号中 "闵氏时空" 替换为 "弯曲时空"，即得广义相对论基本信条：  
  > **(ã) 光子世界线是（所论弯曲时空）类光曲线**  
  > **(b̃) 质点世界线是（所论弯曲时空）类时曲线**  

**核心结论**：  
- 4 维表述 "光子世界线是类光曲线" **直接确定光子运动轨迹**（与坐标无关），而 "坐标光速" 仅反映人为定义的坐标速率；  
- **强烈建议**：在弯曲时空的光传播问题中，应 **避免讨论坐标光速**，转而使用 4 维工具从 $ ds^2 = 0 $ 出发分析。

#### **局域惯性系的常见误解警示**  
弯曲时空虽存在 **局域惯性系**（§6.4），但与闵氏时空的整体惯性系有本质区别：  
- **典型错误**：认为局域惯性系中 "坐标光速等于 $ c $"；  
- **正解**：局域惯性系的坐标光速 **未必等于 $ c $**（因坐标系选择任意性），其与整体惯性系的核心差异在于 **时空曲率的存在**。

### §6.3 等效原理：引力的几何化动机  

对经验定律 $ m_g = m_i $（引力质量等于惯性质量）的深刻思考，是爱因斯坦创立广义相对论的核心动机。为理解其物理图景，先通过 **牛顿力学框架** 分析爱因斯坦电梯思想实验：  

#### **自由下落电梯实验（牛顿力学视角）**  
- **场景**：电梯缆绳断裂自由下落，内部观者失重；  
- **力学分析**：  
  - 苹果受重力 $ m_g \mathbf{g} $（$ \mathbf{g} $ 为重力场强）与惯性力 $ -m_i \mathbf{g} $（$ \mathbf{g} $ 为非惯性系加速度）；  
  - 因 $ m_g = m_i $，合力为零 → 苹果呈 **随遇平衡**（失重状态）；  
- **等效性**：自由下落电梯内 **一切力学实验** 与远离星球的惯性飞船内实验结果相同 → 此即 **弱等效原理（WEP）**。  

> [!NOTE] 4 维语言的前瞻性  
> 此实验揭示：**"引力场" 可通过非惯性系消除**（电梯内失重），暗示引力可能源于时空几何结构（详见 §6.5）。但牛顿力学仅限力学实验，而广义相对论需推广至 **一切物理实验**。

# Slide 124  
**第6章 广义相对论初步**  

### §6.1 引力的实质是时空的弯曲（续）  

自由质点的世界线必为弯曲时空的测地线（基本假设 (b)），而**时空弯曲的几何结构由物质分布决定**（基本假设 (c) 的深化）：  
> **(c) 时空弯曲的源关系**：时空线元 $ ds^2 $ 的具体形式由物质-能量分布通过爱因斯坦场方程确定（详见 §6.5）。  

当引力场足够弱、质点速率 $ v \ll c $ 时，广义相对论力学的计算结果与牛顿力学近似一致。**牛顿力学是广义相对论在弱场低速极限下的近似**（参见 §6.5 末段及梁灿彬、周彬（2006）上册 §7.8.2），但两者物理图景存在本质差异：  

#### **关键对比：牛顿力学 vs 广义相对论**  
| 场景                | 牛顿力学观点                                                                 | 广义相对论观点                                                                 |  
|---------------------|----------------------------------------------------------------------------|------------------------------------------------------------------------------|  
| **苹果落地**        | 苹果受地球引力获得 **3 维加速度**（非惯性运动）                            | 苹果不受 **4 维力**（$ \mathbf{F}_{\text{gi}} = 0 $），是自由质点；世界线为弯曲时空测地线，**4 维加速度为零**（$ \mathbf{A} = 0 $） |  
| **苹果静止于地面**  | 地球引力与支撑力 $ f_{\text{支}} $ 平衡，**3 维加速度为零**（惯性运动）    | 仅受 4 维支撑力 $ \mathbf{F}_{\text{支}} \neq 0 $，世界线 **非测地线**，**4 维加速度非零** |  
| **读者静坐阅读**    | —                                                                          | 世界线 **不重合于弯曲时空测地线**，4 维加速度 $ \mathbf{A} \neq 0 $，未处于最自然运动状态 |  
| **地球绕太阳公转**  | 太阳引力充当向心力，地球有 **3 维加速度**（加速运动）                      | 地球不受 **4 维力**（$ \mathbf{F}_{\text{gi}} = 0 $），沿弯曲时空测地线（绕太阳世界面的螺旋线）运动，**4 维加速度为零** |  

> [!NOTE] 4 维语言的核心优势  
> 4 维表述（$ \mathbf{F} = m \mathbf{A} $）比 3 维表述（$ f = ma $）更本质：  
> - **光子世界线必为类光曲线**（$ ds^2 = 0 $），**质点世界线必为类时曲线**（$ ds^2 > 0 $）；  
> - 弯曲时空中不存在整体惯性系，故 **"坐标光速" 无普适物理意义**（可 $ \neq c $，甚至 $ > c $）；  
> - 应始终采用 **"世界线几何性质"** 而非坐标速率描述运动（详见 §6.2）。  

将引力归结为时空弯曲是人类智慧的伟大突破。黎曼（1854 年）虽提出内禀曲率（黎曼曲率）概念并尝试统一电力与引力，却因 **仅关注 3 维空间曲率而忽略 4 维时空曲率** 未获成功。直至 1905 年狭义相对论问世，时空才被统一（1908 年闵可夫斯基首次明确时空绝对概念），最终在爱因斯坦手中确立 **"引力的实质是时空曲率"** 这一划时代思想。

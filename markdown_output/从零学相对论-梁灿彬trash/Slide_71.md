### §4.2 钟慢效应  

#### 4.2.5 钟慢尺缩效应的实验验证（续）  

但 $K'$ 系认为尺子在高速运动，由尺缩效应可知动尺长只有（仍见图 4-15）  
\[
\Delta x' = \gamma^{-1} \Delta x = 600  \text{m}.
\]  

可见，虽然 $\mu$ 子认为自己只存活了 $2\,\mu\text{s}$（而且出生时遇到尺头 $C_1$），但死亡（衰变）时恰能遇到尺尾（地面）。就是说，$\mu$ 子认为，虽然地面在 $2\,\mu\text{s}$ 内向着自己只走了 $600\,\text{m}$，但这正好就是（缩短后的）尺长，所以地面能在自己死亡前及时赶到自己身边。有兴趣的读者不妨补画以 $\mu$ 子系 $K'$ 为基准系的时空图，由于接近光速，尺子头、尾的世界线将异常靠近。  

可见，$\mu$ 子实验对钟慢、尺缩效应都给出令人信服的验证，是非常成功的“一箭双雕”的事例。  

#### [选读4-2] 宇宙线 $\mu$ 子实验的完整分析  

然而以上只是为教学法目的而设的简化讨论。在宇宙线的实验中无法对 $\mu$ 子做单个跟踪测量，只能对处于某一速率段内的一批 $\mu$ 子进行观测。作为放射性粒子，$\mu$ 子的衰变服从指数衰变律：  
\[
n = n_0 e^{-t / \tau_0}, \tag{4-2-19}
\]  
其中 $n$ 和 $n_0$ 分别代表 $t$ 秒和 $0$ 秒时的 $\mu$ 子数，$\tau_0$ 是常数。上式表明当 $t = \tau_0$ 时 $n = n_0 / e$，即 $\mu$ 子数减少至初始值的 $1/2.7$ 倍，故 $\tau_0$ 称为 $\mu$ 子的**平均寿命**（average lifetime）。需注意：平均寿命与**半衰期**（粒子数减半所需时间）不同。  

式 (4-2-19) 是狭义相对论问世前的实验公式，适用于静止放射性物质。对于高速 $\mu$ 子，以 $K$ 和 $K'$ 分别代表地球系和 $\mu$ 子静止系。由于 $K'$ 认为 $\mu$ 子静止，该式适用但需改用 $K'$ 系时间 $t'$：  
\[
n = n_0 e^{-t' / \tau_0}, \quad \text{（$n$ 和 $n_0$ 为 $t'$ 秒和 $0$ 秒时的 $\mu$ 子数）}
\]  
令 $\mu$ 子相对地球速率为 $u$，$\gamma \equiv \frac{1}{\sqrt{1 - u^2 / c^2}}$，由钟慢效应 $t' = t / \gamma$，代入得  
\[
n = n_0 e^{-t / (\gamma \tau_0)}, \tag{4-2-20}
\]  
可见 $\gamma \tau_0$ 才是运动 $\mu$ 子的平均寿命。  

$\mu$ 子在高空以 $u > 0.99c$ 向地球运动，总数因衰变而减少。Frisch 和 Smith (1963) 选择华盛顿山（Mt. Washington，位于美国新罕布什尔州）山顶实验室和哈佛大学地面实验室作为测量点，分别记录速率介于 $0.9950c$ 和 $0.9954c$ 之间的 $\mu$ 子数，对应式 (4-2-20) 中的 $n_0$ 和 $n$。

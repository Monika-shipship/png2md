# Slide 230

量的等式（quantity equation）直接描述这3个量的关系。为了明确区分量和数以及区分量的等式和数的等式，本小节分别以大写字母代表量、小写字母代表数。于是数的等式 $w = f l$ 相应的量的等式为  
$$W = F L \quad \text{(A-2')}$$  

既然用不同单位测同一量得数不同，反映同一物理规律的数的等式就可能因单位改变而改变。例如，若力的单位由 N 改为 dyn 而功和距离的单位不变，式 (A-2) 需修正。以 $f_{\text{N}}$ 及 $f_{\text{dyn}}$ 分别代表以 N 及 dyn 为单位测同一力 $F$（作为量）所得的数，由于  
$$1\ \text{N} = 10^5\ \text{dyn} \quad \text{(量的等式)}$$  
由式 (A-1$'$) 得  
$$\frac{f_{\text{N}}}{f_{\text{dyn}}} = \frac{\text{dyn}}{\text{N}} = 10^{-5}$$  
即  
$$f_{\text{N}} = 10^{-5} f_{\text{dyn}}$$  
因式 (A-2) 中的 $f$ 是以 N 为单位测力所得的数，可明确表为  
$$w = f_{\text{N}} l$$  
代入 $f_{\text{N}} = 10^{-5} f_{\text{dyn}}$ 得  
$$w = 10^{-5} f_{\text{dyn}} l$$  
即若以 J、dyn 及 m 为单位测量功、力及距离，则数的关系为  
$$w = 10^{-5} f l \quad \text{(A-3)}$$  

可见同一物理规律在不同单位制下的数值表达式仅差一个附加因子（当等式含多項时，各項因子可不同）。若在表达式右侧补单位依赖的比例系数 $k$，则式子对任意单位成立。  

单位选择本属任意，但若对每类量独立选单位（“一盘散沙”），物理规律的数值表达式比例系数将过于复杂。为简化表达式，应采用单位制。单位制由以下要素构成：  
(1) 选定若干基本量并独立指定其单位（基本单位）；  
(2) 非基本量称为导出量，其单位（导出单位）由基本单位按规则导出：  
　　- 找出联系导出量 $A$ 与基本量的物理规律；  
　　- 写出该规律的数值表达式；  
　　- 指定比例系数 $k=1$，即定义 $A$ 的导出单位。  

> [!NOTE] 🖼️ Figure 描述  
> 图形示意单位制构建逻辑：左侧为基本量集合（长度、质量、时间），右侧为导出量网络（速度、力等），箭头标注定义式如 $v = l/t$。核心强调 $k=1$ 的指定如何将基本单位“传递”至导出单位，形成层级化单位体系。几何结构呈树状分形，突出国际单位制（SI）的自洽性。

单位制族中导出单位随基本单位改变的依从关系，需通过量纲概念描述（详见 A.1.2 节）。

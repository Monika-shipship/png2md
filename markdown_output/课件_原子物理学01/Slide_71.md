# 电子的发现

## (4) J.J. Thomson 的荷质比 $q/m$ 测量

$E \neq 0, B = 0$ 撤掉磁场，只加电场，射线向上偏转，偏转量为 $h$。

经过偏转板，粒子获得垂直速度：$v_{\perp} = at = \frac{qE}{m} \frac{l}{v_{\parallel}} = \frac{qlB}{m}$

所以 $\frac{h}{L} = \frac{v_{\perp}}{v_{\parallel}} = \frac{qlB}{m} / \frac{E}{B} = \frac{qlB^2}{mE}$ $\implies \frac{q}{m} = \frac{hE}{lLB^2} = \frac{hV}{dlLB^2}$

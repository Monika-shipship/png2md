# 电子的电荷量（基本电荷$e$的测量）

1899年，Thomson和他的学生汤森德和威尔逊利用威尔逊刚刚开始研制的云室，测量了基本电荷$e$。

C. Wilson (1869-1959)  
J.S. Townsend (1868-1957)

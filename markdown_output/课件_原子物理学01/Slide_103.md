# 原子模型：α粒子散射实验和卢瑟福原子模型

由角动量守恒

$$L = mv_0 b = mr^2 \frac{d\varphi}{dt}$$

→ $$\frac{1}{r^2} = \frac{1}{v_0 b} \frac{d\varphi}{dt}$$

↓ 代入

$$dv_y = \frac{F_y}{m} dt = \frac{1}{4\pi\varepsilon_0} \frac{2Ze^2}{r^2} \sin\varphi \frac{1}{m} dt$$

→ $$dv_y = \frac{2Ze^2}{4\pi\varepsilon_0} \frac{\sin\varphi}{mv_0 b} d\varphi$$

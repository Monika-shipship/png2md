# 原子模型：α粒子散射实验和卢瑟福原子模型

[例1.3] 设H原子中电子绕原子核作圆周运动，原子的初始半径为 $10^{-10}\ \text{m}$，请由经典电磁理论估计电子落到原子核上的时间。

[解] 由拉莫尔公式，一个以加速度 $a$ 运动的带电粒子的辐射功率为：
$$P = \frac{dE}{dt} = -\frac{e^2 a^2}{6\pi \varepsilon_0 c^3}$$

而 $\frac{dE}{dt} = \frac{dE}{dr} \frac{dr}{dt}$，由此 $\frac{dr}{dt} = \frac{dE}{dt} \Big/ \frac{dE}{dr}$

由 $E = -\frac{1}{2} \frac{e^2}{4\pi \varepsilon_0 r}$，得 $\frac{dE}{dr} = \frac{1}{2} \frac{1}{4\pi \varepsilon_0} \frac{e^2}{r}$

圆周运动的加速度 $a = \frac{v^2}{r} = \frac{e^2}{4\pi \varepsilon_0} \frac{1}{mr^2}$，有 $\frac{dr}{dt} = -\frac{4}{3} \left( \frac{e^2}{4\pi \varepsilon_0} \right)^2 \frac{1}{m^2 r^2 c^3}$

因此 $dt = -\frac{3}{4} \left( \frac{4\pi \varepsilon_0}{e^2} \right)^2 m^2 r^2 c^3 dr$

可得 $t = \int_{R}^{0} dt = \frac{1}{4} \left( \frac{4\pi \varepsilon_0}{e^2} \right)^2 m^2 R^3 c^3 \approx 10^{-10}\ \text{s}$

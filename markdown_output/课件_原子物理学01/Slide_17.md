# 原子的质量与大小

## (3) 原子的大小

单个原子平均占有的空间体积  
$$V_A = \frac{1}{N} = \frac{A}{\rho N_A}$$

球形密堆积  
$$V_A = (2R)(\sqrt{3}R)(\sqrt{8/3}R)$$  
$$= 4\sqrt{2}R^3$$  

$$4\sqrt{2}R^3 = \frac{A}{\rho N_A}$$  

$$R = \left( \frac{A}{4\sqrt{2}\rho N_A} \right)^{1/3}$$

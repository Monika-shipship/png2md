# 原子模型：α粒子散射实验和卢瑟福原子模型

**实际的靶**：原子数密度为$N$，厚度为$t$，α粒子束与靶相交的截面为$S$。

实验上测量的是：对应(单位时间)入射α粒子数目$n$，散射到角度$\theta$方向上，探测器所张的立体角$d\Omega$内的粒子数$dn$。

# 原子模型：α粒子散射实验和卢瑟福原子模型

- 微观相互作用机制
- $\frac{d\sigma}{d\Omega}$
- 宏观可观测量
- $\frac{dn}{n} = \frac{d\sigma}{d\Omega} N t d\Omega$
- $\frac{d\sigma}{d\Omega} = \left( \frac{1}{4\pi\epsilon_0} \right)^2 \left( \frac{2Ze^2}{4E} \right)^2 \frac{1}{\sin^4(\theta/2)}$  
  α粒子与原子的库仑散射过程
- 总散射截面或积分截面  
  $\sigma = \int \frac{d\sigma(\theta)}{d\Omega} d\Omega = 2\pi \int_0^\pi \frac{d\sigma(\theta)}{d\Omega} \sin\theta \, d\theta$

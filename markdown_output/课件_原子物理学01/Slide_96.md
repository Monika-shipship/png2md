# 原子模型：α粒子散射实验和卢瑟福原子模型

**单次散射：** 一个α粒子与一个原子散射(碰撞)一次

**散射角估计：**

α粒子 $p_0$  
+2e  
$b$ (瞄准距离)  

所以：  
$$\Delta p \sim \frac{1}{4\pi\varepsilon_0} \frac{2Ze^2}{R^2} \frac{R}{\sqrt{2E/m_\alpha}} = \frac{1}{4\pi\varepsilon_0} \frac{2Ze^2}{\sqrt{2E/m_\alpha} R}$$

$\Delta p$ 很小，所以有 $\Delta p \sim p_0 \theta_1$

则：  
$$\theta_1 \sim \frac{\Delta p}{p_0} = \frac{\Delta p}{m_\alpha v} = \frac{1}{4\pi\varepsilon_0} \frac{2Ze^2}{\sqrt{2E/m_\alpha} R} \frac{1}{\sqrt{2m_\alpha E}} = \frac{1}{4\pi\varepsilon_0} \frac{Ze^2}{RE}$$

# 原子模型：Thomson模型

电子围绕原子中心作简谐振动，辐射电磁波。

(1) 原子的电中性；  
(2) 稳定性；  
(3) 定性解释辐射特性。

## Figure & Layout Description
页面采用左右分栏布局。左侧区域包含一个三维立体原子示意图：主体为粉红色半透明球体（直径约占页面宽度1/3），球体内部嵌入9个金黄色球形电子（表面带有"−"符号），球体中心有淡紫色十字形结构代表正电荷分布。右侧区域为几何图形阵列，按3列4行排列共12组图形（编号2-12），每组包含多边形轮廓（三角形至十二边形）及内部点/线结构：第1行包含编号2（线段）、3（三角形）、4（正方形）、5a/5b（五边形）；第2行包含6a/6b（六边形变体）、7（七边形）；第3行包含8a-8c（八边形多电子构型）；第4行包含9-12（九至十二边形）。所有几何图形均采用黑色细线绘制，部分图形内部有黑点表示电子位置。页面顶部有黑色粗体标题，下方有灰色水平分割线，文字内容使用标准黑体字。

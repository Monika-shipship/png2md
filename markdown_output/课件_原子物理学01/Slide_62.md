# 电子的发现

## (3) 法拉第电解实验

当时，对带电粒子的了解基本上来自电解实验。

阴极 ← $Cu^{2+} + 2Cl^{-}$ → 阳极  
阴极：$Cu^{2+} + 2e^{-} = Cu$  
阳极：$2Cl^{-} - 2e^{-} = Cl_{2} \uparrow$

M. Faraday (1791-1867)

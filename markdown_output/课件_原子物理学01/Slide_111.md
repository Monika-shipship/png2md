## 原子模型：α粒子散射实验和卢瑟福原子模型

α粒子散射到θ角方向上锥面立体角$d\Omega$内的概率  
$$\frac{dn}{n} = \frac{d\Sigma}{S} = \frac{NStd\sigma}{S} = Ntd\sigma$$

散射到θ角方向上单位立体角内的概率  
$$\frac{dn}{nd\Omega} = Nt\frac{d\sigma}{d\Omega} = Nt\left(\frac{1}{4\pi\varepsilon_0}\right)^2 \left(\frac{2Ze^2}{4E}\right)^2 \frac{1}{\sin^4(\theta/2)}$$

## Figure & Layout Description

图像采用左右分栏布局，左侧为主示意图，右侧为局部放大图。顶部有黑色粗体标题"原子模型：α粒子散射实验和卢瑟福原子模型"，下方有深灰色分隔线。

主示意图中：
- 左侧为红色渐变圆柱体（标有$n$），代表入射α粒子束，沿水平轴线向右延伸
- 中间为蓝色长方体靶材（厚度标注$t$），内部包含紫色半透明区域（标有$S$），区域内分布12个蓝色圆圈（代表原子）
- 从靶材右侧延伸出粉色箭头束（散射方向），主箭头标注散射角$\theta$
- 靶材左侧面标注$d\sigma$（微分散射截面）

右侧放大图：
- 显示环形探测器结构，外环为灰色，内环为浅灰色
- 环形区域标注$dn$和$d\Omega$（立体角）
- 两条虚线表示探测器接收角度范围

整体配色以红蓝为主：红色表示粒子束路径，蓝色表示原子和靶材，紫色表示原子核影响区域，粉色表示散射方向。所有文字标注均采用黑色无衬线字体，公式部分使用标准数学符号排版。

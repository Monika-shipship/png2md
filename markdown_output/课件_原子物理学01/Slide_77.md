# 电子的电荷量（基本电荷e的测量）

## 密立根油滴实验

跟踪一个油滴，设油滴半径$r$，密度$\rho$；空气密度$\rho_0$，粘滞系数$\eta$。

① 电容器上不加电压。

**Stokes定律** $F_{\text{阻}} = 6\pi\eta r v$

$v$是下降速度

达到平衡时，油滴达到一个收尾速度$v_g$

$$\frac{4}{3}\pi r^3 \rho g = 6\pi\eta r v_g + \frac{4}{3}\pi r^3 \rho_0 g$$

$$6\pi\eta r v_g = \frac{4}{3}\pi r^3 (\rho - \rho_0) g$$

测量收尾速度$v_g$，得到

$$r = \sqrt{\frac{9\eta v_g}{2(\rho - \rho_0)g}}$$

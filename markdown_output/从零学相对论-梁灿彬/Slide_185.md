§8.5 星光偏转  
类光测地线的计算给出  
\[
\frac{d^2\mu}{d\varphi^2} + \mu = 3M\mu^2 \quad (M \text{ 为太阳质量})。
\tag{8-5-1}
\]

[注：本书并未介绍测地线的数学定义及计算公式，读者只能承认上述结果。] 当 $M = 0$ 时（平直时空），方程（8-5-1）的通解为  
\[
\mu(\varphi) = \frac{1}{l}\sin(\varphi + \varphi_0),
\tag{8-5-2}
\]  
其中 $l$ 和 $\varphi_0$ 为积分常数。约定当 $\varphi = 0$ 时光子在无限远，即 $\mu(0) = 1/r(0) = 0$，则 $\varphi_0 = 0$，故  
\[
\mu(\varphi) = \frac{1}{l}\sin\varphi,
\tag{8-5-3}
\]  
这是 2 维欧氏空间中用极坐标 $r,\varphi$ 表述的直线方程。要看出这一点，取 $r = 0$ 为直角坐标系 $\{x,y\}$ 的原点，则  
\[
x = r\cos\varphi,
\tag{8-5-4}
\]  
\[
y = r\sin\varphi = \frac{1}{\mu}\sin\varphi = l = \text{常数},
\tag{8-5-5}
\]  
其中第三步用到式（8-5-3）。所以光子的空间轨迹是平行于 $x$ 轴、与原点距离为 $l$ 的直线（见图 8-7）。请注意 $r$ 和 $\varphi$ 沿该直线都在变化，不变的是 $y$。由于 $r$ 的取值范围是 $0 < r < \infty$，式（8-5-4）表明 $x$ 的取值范围是 $-\infty < x < \infty$。要讨论星光偏转当然不能取 $M = 0$，但因 $M/r \ll 1$，求一级近似解 $\mu_1(\varphi)$ 便已足够。把式（8-5-3）的 $\mu(\varphi)$ 看作零级近似解 $\mu_0(\varphi)$，代入式（8-5-1）右边便得 $\mu_1(\varphi)$ 所应满足的微分方程：  
\[
\frac{d^2\mu_1}{d\varphi^2} + \mu_1(\varphi) = \frac{3M}{l^2}\sin^2\varphi。
\tag{8-5-6}
\]  
不难验证下式是方程（8-5-6）的解：  
\[
\mu_1(\varphi) = \frac{1}{l}\sin\varphi + \frac{M}{l^2}(1 - \cos\varphi)^2。
\tag{8-5-7}
\]

> 直角坐标系示意图：原点（$r=0$）为坐标交点，$x$ 轴水平向右，$y$ 轴垂直向上。$x$ 轴正半轴标注“来自恒星”（含实心圆点），光子轨迹为过原点的水平直线（标注“光子空间轨迹”），向左延伸至观察者方向（末端箭头标注“走向观察者”）。$y$ 轴正方向标注距离 $l$（从原点垂直延伸的线段），轨迹上标注径向距离 $r$ 和角度 $\varphi$（原点处弧线标注）。

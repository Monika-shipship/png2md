# Slide 4

$$\frac{d^2 \Phi}{d\phi^2} + \lambda \Phi = 0$$

由于周期性 $\Phi(\phi) = \Phi(\phi + 2\pi)$，所以 $\lambda < 0$ 的指数解不行  
$\Phi(\phi)$ 必为三角函数解，$\lambda > 0$ 且 $\lambda = m^2$，$m = 0, 1, 2, \ldots$ 是特征值。  
$$\Phi(\phi) = A \cos m\phi + B \sin m\phi$$

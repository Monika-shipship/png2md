# 光电效应与Einstein的光量子论-光电效应

## 实验现象 III

(1) $V_0$(光电子的最大初始动能)只与入射光频率$\nu$(波长)有关，$\nu$越小，初始动能越小；当$\nu = \nu_0$时，$V_0=0$，初始动能为零；$\nu < \nu_0$，电子不能脱离金属表面。

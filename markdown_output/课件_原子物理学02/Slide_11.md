# 黑体辐射与Planck的量子假说-**黑体辐射**

【维恩公式】：$$M_{0\lambda}(T) = \frac{c_1}{\lambda^5} e^{-\frac{c_2}{\lambda T}}$$  
R-J: $$8\pi k_B T \lambda^{-4}$$  

1900年，普朗克宣称经验公式  

**当$x$很小时，$e^x = 1 + x + ...$**  

$$u(\lambda) = \frac{c_1 \lambda^{-5}}{e^{c_2 / \lambda} -1}$$

Max Planck  
April 23, 1858 – October 4, 1947  
The Nobel Prize in Physics 1918

## H原子模型的检验

Bohr将其归属为He的一价离子的光谱。  
He⁺, Li²⁺, Be³⁺, 都是类氢离子：  
+Ze的原子核与核外的一个电子组成。  

$$E_n = -\frac{1}{2} m_e c^2 \alpha^2 \frac{Z^2}{n^2} = Z^2 E_H, \quad n=1,2,3,\ldots$$

$$r_n = \frac{\hbar c n^2}{Z m_e c^2 \alpha} = \frac{r_H}{Z}, \quad n=1,2,3,\ldots$$

光谱波数  
$$\tilde{\nu} = Z^2 R_{\infty} \left( \frac{1}{m^2} - \frac{1}{n^2} \right)$$

对于He⁺  
$$\tilde{\nu} = 2^2 R_{\infty} \left( \frac{1}{m^2} - \frac{1}{n^2} \right) = R_{\infty} \left( \frac{1}{(m / 2)^2} - \frac{1}{(n / 2)^2} \right)$$

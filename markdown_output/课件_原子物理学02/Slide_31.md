# X射线与Compton效应-X射线的发现

大发明家爱迪生和他的便携式X射线透视仪

## Figure & Layout Description
页面顶部为标题栏，黑色粗体字显示"X射线与Compton效应-"，后接红色粗体字"X射线的发现"，二者间以短横线连接。标题下方有一条深灰色水平分割线。主体内容分为三个区域：
1. 左侧区域：由5张黑白历史照片组成的拼贴图。左上角为X射线管实验装置图（显示真空管与手部照射场景）；左中为鱼的X光影像图；左下为手持锥形X射线装置的示意图；中下为带圆形窗口的X射线设备图；右侧主体为手部骨骼X光片（可见手指骨骼及无名指上的环形异物）。
2. 右侧区域：竖版黑白肖像照，显示身着黑色西装、白色衬衫与领结的爱迪生坐姿像，左手置于扶手，右手轻触头部。
3. 底部区域：居中放置一张黑白照片，展示人物头部伸入大型锥形X射线装置的侧面视角，装置前端可见细长探针结构。
所有图像均为复古颗粒质感的黑白照片，文字说明采用蓝色无衬线字体位于页面最底部。

# 黑体
- **黑体的定义：**
  - 若一物体对什么光都吸收而无反射，我们就称这种物体为“绝对黑体”，简称“黑体”。“绝对黑体”是不存在的
- **黑体的特点：**
  - 黑体与热辐射达到平衡时，单色辐射能量密度$E(\nu, T)$随频率$\nu$变化曲线的形状与位置只与黑体的绝对温度$T$有关，而与空腔的形状及组成的物质无关
- **黑体辐射研究的发展：**
  - 冶金学和天文学等方面的需要，大大推动了对热辐射的研究

# H原子光谱和Bohr原子模型

## H原子的大小

- 中心原子核标注：$+e$
- 轨道参数：
  - $n=1$：$r_1 = a_0$
  - $n=2$：$r_2 = 4a_0$
  - $n=3$：$r_3 = 9a_0$
- 确定的原子大小：$10^{-10} \text{m}$

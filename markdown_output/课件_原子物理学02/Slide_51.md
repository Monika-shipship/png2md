## H原子光谱和Bohr原子模型

### Bohr频率规则

$$
\nu = |E_m - E_n| / h = \left[ \left( -hc \frac{R_H}{n^2} \right) - \left( -hc \frac{R_H}{m^2} \right) \right] / h \quad (\text{设} \, m < n)
$$

$$
= R_H c \left( \frac{1}{m^2} - \frac{1}{n^2} \right) = R_H c \frac{(n + m)(n - m)}{m^2 n^2}
$$

当 $n$ 很大时，邻能级间跃迁的频率 ($n = m+1 \to \infty$)

$$
\nu = R_H c \frac{(n + n - 1)(n - n + 1)}{(n - 1)^2 n^2} \approx R_H c \frac{2}{n^3}
$$

# 黑体辐射与Planck的量子假说-黑体辐射

$$u(\lambda) = \frac{8\pi h c \lambda^{-5}}{e^{hc / \lambda k_B T} - 1}$$

**长波极限：**

设 $x = hc / \lambda k_B T$ 是小量

$$e^{hc / \lambda k_B T} - 1 = 1 + hc / \lambda k_B T + \ldots - 1 \approx hc / \lambda k_B T$$

则

$$u(\lambda) \approx \frac{8\pi h c \lambda^{-5}}{hc / \lambda k_B T} = 8\pi \lambda^{-4} k_B T$$

Rayleigh-Jeans公式

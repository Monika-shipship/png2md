## Figure & Layout Description

页面采用四象限布局结构：  
1. **左上区域**：电子能量损失谱仪结构示意图，以黑色线条绘制。包含两个对称的八边形磁体结构（左侧标注"单色器"，右侧标注"分析器"），中间通过圆形"作用室"连接。作用室左侧连接矩形"电子枪"，右侧连接带PSD标识的探测器模块。所有组件通过细线表示电子路径，整体为黑白线稿风格。  
2. **右上区域**：汞原子能级图，垂直坐标轴标注$E$ (eV) 0-10刻度。图表被垂直线分为左右两栏，左栏标题"singulet"，右栏标题"triplet"。各能级以水平线表示，标注$6^1S_0$, $7^1S_0$等符号，能量值用黑色数字标注（6.70 eV, 4.67 eV等）。底部有254 nm波长箭头从$6^1S_0$指向$6^3P_1$能级。  
3. **左下区域**：Hg元素的电子能量损失谱图，坐标系中横轴为"Energy Loss (eV)"（5-14范围），纵轴为"Dipole Oscillator Strength df/dE (eV⁻¹)"（0-2.0范围）。图中显示10余个尖锐峰，主要峰位用文字标注$6^3P_1$等符号，部分峰标注"x0.1"放大系数。图表左上角有"Hg"标识框。  
4. **右下区域**：EELS工作原理示意图，展示弯曲管道结构。从左至右依次标注"cathode"（阴极）、"mono-chromator"（单色器）、"analyzer"（分析器）和"channeltron"。红色箭头表示电子路径$e^-$，中间有分子结构示意图。底部有黄色矩形基底。  
整体页面标题为黑色粗体，"电子能量损失谱仪"为蓝色字体，图表均采用黑白色调，仅底部EELS说明文字为黑色标准字体。

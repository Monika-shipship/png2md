# H原子光谱和Bohr原子模型

$$\tilde{\nu} = (E_n - E_1) / hc = \frac{m_e \alpha^2 c}{2h} \left( \frac{1}{1^2} - \frac{1}{n^2} \right)$$
$$n = 2,3,4,\ldots$$

比较经验公式
$$\tilde{\nu} = R_H \left[ \frac{1}{1^2} - \frac{1}{n^2} \right]$$

里德伯常数的理论公式
$$R_\infty = \frac{1}{(4\pi\varepsilon_0)^2} \frac{m_e e^4}{4\pi \hbar^3 c} = 1.097,373,156 \times 10^7 \, \text{m}^{-1}$$

光谱实验值
$$R_H = 1.096,775,8 \times 10^7 \, \text{m}^{-1}$$

理论计算与实验值符合得非常好！

## Figure & Layout Description

图像由上下两部分构成：  
1. **上半部分**：黑色背景的发射光谱（emission），显示6条垂直彩色谱线（从左到右：深紫、紫、蓝、青、绿、红），其中红色谱线位于650nm附近。  
2. **下半部分**：连续光谱背景的吸收光谱（absorption），标注"(white light background)"，呈现从400nm（紫色）到700nm（红色）的彩虹色带，其中400-500nm区域有5条黑色吸收线，对应上半部分的发射线位置。  
3. **波长标尺**：底部灰色条带标注400nm、500nm、600nm、700nm刻度，与光谱区域精确对齐。  
4. **文字标注**：  
   - 顶部大标题："H原子光谱和Bohr原子模型"  
   - 光谱区域中央灰色条："Hydrogen line spectrum: Balmer series"  
   - 发射光谱区域标注"emission"，吸收光谱区域标注"absorption"  
5. **表格结构**：3行9列，包含谱线名称（Hα至limit）、波长数值（精确到小数点后1位）、颜色分类（Red/Cyan/Blue/Violet/UV）。

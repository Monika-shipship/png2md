# X射线与Compton效应-Compton效应

## 黑体辐射  
腔中的驻波场振荡模式的能量不连续分布  
$$E = n\varepsilon, \quad n = 0,1,2,3,\dots$$  
$$\varepsilon = h\nu$$

## 光电效应  
传播中的光(电磁波)由空间不连续的光量子(光子)组成  
**光子的能量** $E = h\nu$

## Compton效应  
**光子动量** $p = \hbar k$  
$$p = \frac{h}{\lambda}$$

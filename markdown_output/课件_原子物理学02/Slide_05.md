# 黑体辐射与Planck的量子假说-黑体辐射

## 黑体辐射的波长分布

$R(\lambda)d\lambda$

单位面积上发射波长在$\lambda \sim \lambda+d\lambda$电磁辐射的辐射本领。

【维恩公式】：$M_{0\lambda}(T) = \frac{c_1}{\lambda^5} e^{-\frac{c_2}{\lambda T}}$

其中 $c_1$、$c_2$ 是实验确定的参数。

# 特殊的氢原子体系-里德堡原子

实验室：

电子碰撞激发、双光子或多光子激发等方法。

$n \sim 200$ 里德伯氢原子

## Figure & Layout Description
页面顶部有黑色标题"特殊的氢原子体系-"后接红色加粗字体"里德堡原子"，下方配有一条灰色水平分隔线。左侧区域为能级结构示意图：最底部是基态能级 $3s^2 {}^1S_0$，向上依次排列 $3s4s {}^1S_0$（-2.7 eV）、$3s5s {}^1S_0$、$3s6s {}^1S_0$ 等高能级；右侧对应标注 $3s3p {}^3P_{2,1,0}$（-2.7 eV）、$3s3p {}^1P_1$（4.3 eV）、$3s4p {}^1P_1$、$3s5p {}^1P_1$、$3s6p {}^1P_1$、$3s10p {}^1P_1$（7.62 eV）等能级。能级间有带 $\lambda_1$、$\lambda_2$ 标签的跃迁箭头，其中 $\lambda_1$ 从 $3s^2 {}^1S_0$ 指向 $3s3p {}^3P_{2,1,0}$，$\lambda_2$ 从 $3s4s {}^1S_0$ 指向多个 $3snp {}^1P_1$ 能级。右侧为光谱分布图，横轴标有波长 553.1–554.7 nm，纵轴为强度，显示从 3s28p 到 3s38p 的多个尖锐谱线峰，每个峰顶部标注对应能级（如 3s28p、3s29p、...、3s38p）。页面底部有黑色文字说明激发方法和 $n \sim 200$ 里德伯氢原子。标题中"里德堡原子"用红色突出显示，"实验室："文字为蓝色，其余文字均为黑色。

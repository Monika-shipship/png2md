# H原子光谱和Bohr原子模型

按照Bohr对应原理，在大量子数极限下，量子计算与经典计算一致

有  
$$\frac{e}{2\pi} \sqrt{\frac{1}{4\pi\varepsilon_0 m_e r^3}} = R_H c \frac{2}{n^3} \quad \rightarrow \quad r = \sqrt[3]{\frac{1}{4\pi\varepsilon_0} \frac{e^2}{16\pi^2 R_H^2 c^2 m_e}} n^2$$

又  
$$r_n = \frac{1}{4\pi\varepsilon_0} \frac{e^2}{2 R_H h c} n^2$$

得到  
$$R_H = \frac{m_e e^4}{(4\pi\varepsilon_0)^2 4\pi h^3 c}$$

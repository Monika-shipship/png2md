# H原子模型的检验

## 玻尔氢原子理论

① 解释了H原子的大小；  
② 完美地解释了H原子光谱；  
③ 理论上精确计算了里德伯常数。

### 问题一

1897年，天文学家毕克林在观察船尾座ζ星(中国古天文名：弧矢增二十二)的星光光谱时发现了一个新的线系，称为**毕克林线系**。

![光谱图](横坐标为波数刻度，范围约15000-25000 cm⁻¹，标注有$H_\alpha$、$H_\beta$、$H_\gamma$、$H_\delta$等谱线，左侧有"谱系限"标注)

$$\tilde{v} = R' \left( \frac{1}{2^2} - \frac{1}{k^2} \right)$$ 其中，$k = 2.5, 3, 3.5, 4, \dots$

一直被当作H原子的一个光谱线系

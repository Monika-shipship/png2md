# 黑体辐射与Planck的量子假说-黑体辐射

太阳的黑体辐射谱

## Figure & Layout Description
幻灯片顶部有主标题"黑体辐射与Planck的量子假说-黑体辐射"，其中"黑体辐射"以红色粗体字突出显示，其余文字为黑色粗体。标题下方有一条细灰色水平分隔线。

页面主体分为三个主要区域：
1. **中上部左侧区域**：展示一个三维长方体黑体辐射模型，背景为纯黑色。长方体表面呈现从左到右的连续颜色渐变：左侧为深灰色（低温区），中间过渡为红色，右侧逐渐变为黄色至亮白色（高温区），直观表现温度升高时黑体颜色变化。
2. **中上部右侧区域**：垂直排列的温度-颜色对照条，从上至下依次标注"1000K"（深红色）、"2000K"（橙红色）、"3000K"（橙色）、"4000K"（浅橙色）、"5000K"（米黄色）、"6000K"（浅黄色）、"7000K"（浅蓝白色）、"8000K"（浅蓝色）、"9000K"（蓝色）、"10000K"（深蓝色），颜色随温度升高从红到蓝连续渐变，形成完整的色温标尺。
3. **底部区域**：太阳的黑体辐射谱曲线图，横轴标注"Wavelength (nanometers)"（范围350-900nm），纵轴标注"Intensity (counts)"（范围0-4000）。蓝色实线显示太阳辐射强度分布，曲线上用黑色箭头标注多个吸收线位置：K(390nm)、H(395nm)、G(430nm)、F(485nm)、bE(525nm)、D₂+D₁(589nm)、B(625nm)、H₂O(720nm)、A(760nm)等。图表下方有蓝色文字标注"太阳的黑体辐射谱"。

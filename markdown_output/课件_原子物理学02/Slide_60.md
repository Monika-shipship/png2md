# H原子光谱和Bohr原子模型

## H原子的能级和能级图

### H原子的能级

$n \to \infty$时，$E_n \to 0$

$$E = -\frac{1}{2} \frac{e^2}{4\pi\varepsilon_0 r}$$

$r \to \infty$

电子脱离原子核库仑场的束缚，  
**成为自由电子，称为电离**。

自由电子可以具有一定的动能$T$，  
而$r \to \infty$时，势能$V(r) = 0$，所以  
总能量$E = T > 0$，轨道是双曲线的一支，  
动能可以为任意值，不是量子化的。

能级图上是$n \to \infty$时，$E_n \to 0$之上  
连续的正能量区。

$$E_n = -\frac{1}{2} m_e \alpha^2 c^2 \frac{1}{n^2}$$

- 黑色文字："r → ∞"
- 一段黑色文字描述："电子脱离原子核库仑场的束缚，"后接红色粗体文字"成为自由电子，称为电离。"
- 一段黑色文字描述："自由电子可以具有一定的动能T，而r → ∞时，势能V(r) = 0，所以总能量E = T > 0，轨道是双曲线的一支，动能可以为任意值，不是量子化的。"

右侧区域：
- 顶部是一个黄色边框、浅黄色填充的矩形框，内含公式：$E_n = -\frac{1}{2} m_e \alpha^2 c^2 \frac{1}{n^2}$
- 其下是一个能级图，图表标题为"Level energy / eV"，纵轴表示能量（单位eV），从-13.6到0
- 横轴表示主量子数n，从1到∞
- 能级线：水平线表示不同能级，n=1对应-13.6 eV，n=2对应-3.45 eV，n=3对应-1.5 eV，n=4对应-0.9 eV，n=5、6、7和∞依次向上
- 电离极限：n=∞处有一条虚线，标注"ionization limit"
- 光谱系列跃迁：用箭头表示不同系列的跃迁
  - Lyman series：从n>1到n=1的跃迁，黑色箭头指向n=1能级
  - Balmer series：从n>2到n=2的跃迁，红色箭头指向n=2能级
  - Paschen series：从n>3到n=3的跃迁，黑色箭头指向n=3能级
  - Brackett series：从n>4到n=4的跃迁，黑色箭头指向n=4能级
  - Pfund series：从n>5到n=5的跃迁，黑色箭头指向n=5能级

# H原子光谱和Bohr原子模型

常温T=300K

$\frac{N_2}{N_1} \sim \frac{\exp(-E_2 / k_B T)}{\exp(-E_1 / k_B T)} = e^{-(E_2-E_1)/k_B T}$

Boltzmann常数 $k = 8.62 \times 10^{-5} \, \text{eV·K}^{-1}$

$E_1 = -13.6 \, \text{eV}$ \quad $E_2 = \frac{-13.6 \, \text{eV}}{2^2} = -3.4 \, \text{eV}$

$\frac{N_2}{N_1} \sim e^{-(-3.4 \, \text{eV} + 13.6 \, \text{eV}) / (8.62 \times 10^{-5} \, \text{eV·K}^{-1} \cdot 300 \, \text{K})} = e^{-394.6} \sim 4.2 \times 10^{-172} = 0$

常温下，一团冷气体的原子几乎全部处在基态。白光通过，一些H原子吸收光子跃迁到激发态。

只有莱曼线系 $h c \tilde{\nu} = E_n - E_1 = -\frac{1}{2} m_e \alpha^2 c^2 \frac{1}{n^2} + \frac{1}{2} m_e \alpha^2 c^2 \frac{1}{1^2}$

# 实物粒子的波动性–波粒二象性

**简单的数学：**

遮上缝2，电子穿过缝1：  
电子在屏上的行为用复函数 $\phi_1(x)$ 来表示，其分布概率 $P_1 = |\phi_1|^2$

遮上缝1，电子穿过缝2：  
电子在屏上的行为用复函数 $\phi_2(x)$ 来表示，其分布概率 $P_2 = |\phi_2|^2$

双缝打开：其分布概率 $P_{12} = |\phi_1 + \phi_2|^2$

> 电子这样的行为很难用复杂的运动轨迹解释。

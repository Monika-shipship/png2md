# 一维定态问题-无限深方势阱

按照不确定关系：

$$
\Delta p_x \Delta x \ge \frac{\hbar}{2}
$$

因 $\Delta x \sim a$

有 $\Delta p_x \sim \frac{\hbar}{2a} \ne 0$

→ $$
E = \frac{p_x^2}{2m} \sim \frac{\Delta p_x^2}{2m} = \frac{\hbar^2}{8ma^2} \ne 0
$$

## Figure & Layout Description

页面整体采用左右分栏布局。左侧为文字与公式推导区域，包含标题、不确定关系公式及能量推导过程；右侧为势能分布示意图。示意图采用直角坐标系，横轴为位置 $x$（箭头向右），纵轴为能量 $E$（箭头向上）。势阱结构由三个区域组成：左侧 $x < 0$ 和右侧 $x > a$ 区域为浅灰色填充的势垒（barrier），标注 $V(x)=\infty$；中间 $0 \le x \le a$ 区域为白色势阱（well），标注 $V(x)=0$。坐标轴在 $x=0$ 和 $x=a$ 处有明确刻度标记，势垒区域高度延伸至纵轴顶端表示无限高。左侧关键公式旁有蓝色右向箭头指示能量表达式的推导路径，箭头宽度约为公式高度的1.5倍。

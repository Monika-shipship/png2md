# 薛定谔方程-方程的建立

对于非相对论的自由粒子

$$E = \frac{p^2}{2m}$$

于是

$$i\hbar \frac{\partial \psi}{\partial t} = E\psi = \frac{p^2}{2m}\psi = -\frac{\hbar^2}{2m} \nabla^2 \psi$$

$$\nabla^2 \psi = -\frac{p^2}{\hbar^2} \psi$$

得

$$i\hbar \frac{\partial \psi}{\partial t} = -\frac{\hbar^2}{2m} \nabla^2 \psi$$

这是自由粒子满足的微分方程

# 一维定态问题-无限深方势阱

**结论：**

(2) 粒子在一维无限深势阱中的概率(密度)分布

$$|u(x)|^2 = \begin{cases}
\frac{2}{a}\sin^2\frac{n\pi}{a}x, & 0 \leq x \leq a \\
0, & x < 0, or, x > a
\end{cases}$$

$$n = 1,2,3,\ldots$$

概率分布不均匀，存在概率为零的节点。

但：**概率分布不随时间变化！**

# 波函数及其统计解释–态叠加原理

## Knowing which way (测量)

电子被1看到，表明测量到电子处在 $\psi_1(x)$ 态，  
测量使处在叠加态的电子坍缩到 $\psi_1(x)$ 态。  
此时电子出现在 $x$ 点的概率为 $P_1 = |\psi_1(x)|^2$

电子被2看到，表明测量到电子处在 $\psi_2(x)$ 态，  
测量使处在叠加态的电子坍缩到 $\psi_2(x)$ 态。  
此时电子出现在 $x$ 点的概率为 $P_2 = |\psi_2(x)|^2$

电子没有被看到，电子仍处在 $\psi_1(x) + \psi_2(x)$ 的叠加态，  
此时电子出现在 $x$ 点的概率为 $P_{12} = |\psi_1 + \psi_2|^2$

# 一维定态问题-方势垒散射

## 区域(II)：$0 < x < a$, $V(x) = V_0$

$$\left(-\frac{\hbar^2}{2m}\frac{d^2}{dx^2} + V_0\right)u = Eu$$

令 $k_2 = \frac{\sqrt{2m(V_0 - E)}}{\hbar}$

则 $\frac{d^2u}{dx^2} = k_2^2 u$

解为 $u(x) = A_2 e^{k_2 x} + B_2 e^{-k_2 x}$

## 区域(III)：$x > a$, $V(x) = 0$

$$-\frac{\hbar^2}{2m}\frac{d^2u}{dx^2} = Eu$$

解为 $u(x) = A_3 e^{ik_1 x} + B_3 e^{-ik_1 x}$

**透射波** ← 绿色箭头  
**无 -x方向的平面波** ← 绿色箭头  
**$B_3 = 0$** ← 蓝色箭头

## 波函数及其统计解释–波函数的引入

### 经典波
**弦振动**  
$$\frac{\partial^2 y}{\partial x^2} = \frac{1}{v^2} \frac{\partial^2 y}{\partial t^2}$$  
其中相速度 $v = \sqrt{T / \rho_l}$  
$y(x,t)$ 波函数  

**谐波解**  
$y(x,t) = y_0 \cos(kx - \omega t)$  
或 $y(x,t) = y_0 \sin(kx - \omega t)$  
或 $y(x,t) = y_0 e^{i(kx - \omega t)}$  
其中 $k = 2\pi / \lambda$  
$\omega = 2\pi f$  
$y(x,t)$ 是位移  

### 电磁波
**电场**  
$$\frac{\partial^2 E}{\partial x^2} = \frac{1}{c^2} \frac{\partial^2 E}{\partial t^2}$$  

**谐波解(平面单色波)**  
$E(x,t) = E_0 \cos(kx - \omega t)$  
或 $E(x,t) = E_0 e^{i(kx - \omega t)}$  

**波函数：实际的物理量**

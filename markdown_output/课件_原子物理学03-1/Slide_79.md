# 一维定态问题-隧道效应的应用

## Figure & Layout Description
页面分为左右两个主要图示，背景为纯白色。左侧为STM设备结构示意图，右侧为能带与隧道效应原理图。

左侧示意图：
- 整体为黑色边框矩形框，内部包含：底部蓝色"specimen"样品基座，其上放置黑色"tip"探针，探针连接黄色圆柱形"piezo element"压电元件（标注绿色双向箭头表示可调节位移）。
- 左侧有电流表"A"与电压表"V"符号，连接探针与样品。
- 右上角有圆形放大区域，显示探针（绿色球体阵列）与样品（蓝色球体阵列）的原子级接触面，红色虚线标注"trajectory"表示探针扫描路径，绿色箭头指示"tip"移动方向。

右侧原理图：
- 由两个蓝色矩形块组成，左侧块标注"Sample"，右侧块标注"Tip"。
- 样品块顶部标注费米能级$E_F$（蓝色虚线），上方绘制蓝色波浪形势垒曲线，曲线峰值处标注势垒高度$\Phi$（黑色双向箭头）。
- 样品与探针之间有水平蓝色箭头表示电子隧穿方向，下方标注偏置电压"eV"（黑色双向箭头）。
- 电子波函数以蓝色实线波形表示，从样品穿过势垒延伸至探针区域。

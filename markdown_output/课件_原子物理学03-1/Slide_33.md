# Slide 33

# 实物粒子的波动性–波粒二象性

## Knowing which way

- 左侧示意图：光子 $h\nu$ 入射到双原子分子（间距 $R_0$），发射电子 $e^-$ 的轨迹满足 $R \sin\theta$ 条件，传播距离 $D$ 后形成干涉图样
- 右侧 3×3 子图矩阵：
  - a/d/g：实验结果（Experiment）热力图
  - b/e/h：理论模拟（Theory）热力图
  - c/f/i：实验与理论数据对比曲线（红色实线为实验，蓝色虚线为理论）
- 左下角双缝实验装置示意图（a-d）：
  - a/c：双缝装置结构（含可调节挡板）
  - b/d：电子自旋态示意图（红色箭头表示自旋方向）
- 文字标注：
  - "Einstein–Bohr recoiling double-slit gedanken experiment performed at the molecular level"
  - "Nature Photonics 9, 120 (2015)"

## Figure & Layout Description

页面采用三栏式布局：  
1. **左侧上部**：包含分子干涉示意图。蓝色波浪线与箭头表示入射光子 $h\nu$，中心为双原子分子结构（两个浅绿色球体，标注 $R_0$），红色虚线轨迹显示电子 $e^-$ 发射路径，满足 $R \sin\theta$ 几何条件。下方标注传播距离 $D$，右侧黑色竖线代表探测屏，叠加黑色波形图展示干涉条纹。  
2. **左侧下部**：双缝实验装置示意图。  
   - a/c：蓝色挡板构成双缝结构，a显示对称光路，c显示不对称光路（含黄色调节部件）  
   - b/d：电子自旋态图示，b为自旋向上（红色向上箭头），d为自旋反向（绿色向下箭头）  
3. **右侧区域**：3行3列子图矩阵（a-i）：  
   - 每行对应不同实验条件：第一行（a-c）为 $4p$ molecular，第二行（d-f）为 $u^+$ molecular，第三行（g-i）为 $u^+$ atomic  
   - 每列功能：左列（a/d/g）为实验热力图（红-黄-绿-蓝渐变表示强度），中列（b/e/h）为理论模拟图，右列（c/f/i）为实验（红点）与理论（蓝线）的计数曲线对比  
   - 所有子图横轴标注 $\cos\theta$ 或 $m\theta$，纵轴标注电子能量（eV）  
4. **文字元素**：  
   - 顶部标题为黑体中文，英文副标题 "Knowing which way" 为蓝色  
   - 底部引用文字为黑色常规字体，期刊名称 "Nature Photonics" 加粗

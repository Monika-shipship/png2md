# 实物粒子的波动性–波粒二象性

> 对于单个电子：或者穿过缝1，或者穿过缝2 (子弹的图像)

屏上探测到的电子分成两类：
- 穿过缝1的电子
- 穿过缝2的电子

屏上探测到的分布是穿过缝1的电子和穿过缝2的电子的总效果

遮上缝2： $P_1$ (穿过缝1的概率)  
遮上缝1： $P_2$ (穿过缝2的概率)  

双缝打开： $P_{12} \neq P_1 + P_2$  
存在干涉！

## Figure & Layout Description
页面顶部是黑色粗体标题"实物粒子的波动性–波粒二象性"，下方有一条深灰色水平分隔线。主体内容分为左右两部分：左侧为实验示意图，右侧为概率分布曲线图。  
- **左侧示意图**：  
  - 最左侧有一个红色实心箭头（代表电子源）指向右方，箭头右侧有三个红色虚线箭头表示运动方向。  
  - 中间是两条平行的蓝色垂直线，分别标注数字"1"（上方）和"2"（下方），代表双缝。  
  - 右侧是一个蓝色垂直线标记为"S"（屏幕），其上方有向上的蓝色x轴箭头。  
- **右侧曲线图**：  
  - 包含三个子图，从左到右依次标记为(a)、(b)、(c)。  
  - (a)是绿色弯曲曲线，底部标注"$P_1$"；(b)是绿色弯曲曲线，底部标注"$P_2$"；(c)是红色波浪形曲线，底部标注"$P_{12}$"。  
  - 所有曲线均从屏幕S的紫色矩形源点（代表电子发射位置）出发。  
- **文本与箭头元素**：  
  - 标题下方有蓝色右向箭头引导文字："对于单个电子：或者穿过缝1，或者穿过缝2 (子弹的图像)"。  
  - 中部有橙色双向箭头连接文字："屏上探测到的电子分成两类：" 与右侧两个选项"穿过缝1的电子"、"穿过缝2的电子"。  
  - 橙色箭头下方有一个大型蓝色向下箭头，指向文字"屏上探测到的分布是穿过缝1的电子和穿过缝2的电子的总效果"。  
  - 底部左侧列出"遮上缝2： $P_1$ (穿过缝1的概率)"和"遮上缝1： $P_2$ (穿过缝2的概率)"，右侧有一个黄色圆角矩形框，内含文字"双缝打开： $P_{12} \neq P_1 + P_2$"和"存在干涉！"，其中"存在干涉！"为红色字体。  
- **颜色与层级**：  
  - 主标题黑色，分隔线深灰色，电子源箭头红色，双缝和x轴蓝色，概率曲线(a)(b)绿色、(c)红色，文本主要为黑色，关键结论用黄色背景和红色字体突出。  
  - 布局层次分明：顶部标题→中间图示→底部逻辑推导，箭头元素引导阅读顺序，强调从单缝到双缝的对比逻辑。

# 实物粒子的波动性–波粒二象性

- 对于单个电子：或者穿过缝1，或者穿过缝2（子弹的图像）  
- 屏上探测到的电子分成两类：  
  - 穿过缝1的电子  
  - 穿过缝2的电子  
- 屏上探测到的分布是穿过缝1的电子和穿过缝2的电子的总效果  
- 遮上缝2：$P_1$（穿过缝1的概率）  
- 遮上缝1：$P_2$（穿过缝2的概率）  
- 双缝打开：$P_{12} = P_1 + P_2$？

# 一维定态问题-无限深方势阱

驻波解：

$$ a = n \frac{\lambda}{2} = n \frac{h}{2p} $$

$$ p = \frac{h}{2a} n $$

$$ E = \frac{p^2}{2m} = \frac{h^2}{8ma^2} n^2 = \frac{\pi^2 \hbar^2}{2ma^2} n^2, \quad n=1,2,3,\ldots $$

## Figure & Layout Description
页面顶部有黑色粗体标题"一维定态问题-无限深方势阱"，下方有一条细长的深灰色水平分隔线。主体内容分为左右两部分：左侧为公式推导区域，右侧为2×3网格排列的波形图。左侧区域包含"驻波解："文字说明，其下方有三组公式：第一组公式$a = n \frac{\lambda}{2} = n \frac{h}{2p}$；第二组公式$p = \frac{h}{2a} n$左侧有一个蓝色右向箭头；第三组公式$E = \frac{p^2}{2m} = \frac{h^2}{8ma^2} n^2 = \frac{\pi^2 \hbar^2}{2ma^2} n^2, \quad n=1,2,3,\ldots$左侧也有一个蓝色右向箭头。右侧网格图包含6个子图（A-F），每个子图由两条垂直黑线（代表势阱边界）和内部波形组成：A子图中央有一个实心红点；B子图有一条蓝色正弦曲线，从左边界0点开始，形成单峰后回到右边界0点（半波）；C子图有一条蓝色正弦曲线，形成双峰结构（完整波）；D子图有一条蓝色正弦曲线，形成三峰结构；E子图包含两条曲线，蓝色曲线为双峰结构，红色曲线为单峰结构，二者部分重叠；F子图包含两条曲线，蓝色曲线为三峰结构，红色曲线为单峰结构，二者部分重叠。所有子图的垂直边界线高度一致，波形均在边界处归零。页面背景为纯白色，文字和公式均为黑色，箭头为蓝色，波形图中A子图的点为红色，其余波形为蓝色（E和F含红色曲线）。

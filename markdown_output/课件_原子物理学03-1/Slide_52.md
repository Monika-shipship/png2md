# 薛定谔方程-方程的建立

一般情形下，粒子在外场 $V(\mathbf{r}, t)$ 中运动，则在非相对论的情况下

$$E = \frac{p^2}{2m} + V(\mathbf{r}, t)$$

两边同乘 $\psi$:
$$E\psi = \left[ \frac{p^2}{2m} + V(\mathbf{r}, t) \right] \psi$$

类比:
$$i\hbar \frac{\partial \psi}{\partial t} = E\psi$$
$$\nabla^2 \psi = -\frac{p^2}{\hbar^2} \psi$$

则有：

薛定谔方程

$$i\hbar \frac{\partial \psi}{\partial t} = \left[ -\frac{\hbar^2}{2m} \nabla^2 + V(\mathbf{r}, t) \right] \psi$$

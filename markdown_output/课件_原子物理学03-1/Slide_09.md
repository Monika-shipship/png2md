## Figure & Layout Description
页面整体为白色背景，采用左右分栏布局。顶部有黑色粗体主标题“实物粒子的波动性–德布罗意的物质波假设”，标题下方有一条细长的深灰色水平分割线。左侧区域包含三部分垂直排列的内容：最上方是蓝色文字“氢原子的驻波解释”；中间是红色粗体文字“Standing Waves in a string”，其正下方有一条细红色水平线段，线段上均匀分布四个红色小圆点；最下方是黑色公式文本“L = n · λ/2”。右侧区域垂直排列三个驻波示意图，每个图均包含以下元素：左侧有黑色标签“n = [数字]”，右侧有垂直的深灰色边界线表示弦的固定端点。具体图示细节如下：
- 顶部图（n=1）：标注“Fundamental”，上方有水平双箭头线标注“String length”。波形由上半部分紫色弧线和下半部分绿色弧线组成，形成一个完整的半波（一个波腹），无节点。
- 中间图（n=2）：标注“Node”，波形由两个连续的半波组成（紫色和绿色交替），在中心位置有一个黑色标注“Node”指向波形交点（节点）。
- 底部图（n=3）：标注“Nodes”和“Overtones”，波形由三个连续的半波组成（紫色和绿色交替），在两个交点处有黑色标注“Nodes”指向节点位置。
所有波形图均采用紫色（上半波）和绿色（下半波）填充，边界线为深灰色。文字颜色严格区分：主标题为黑色，氢原子解释为蓝色，驻波标题为红色，公式和图示标签为黑色。

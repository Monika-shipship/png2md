# 实物粒子的波动性–波粒二象性

## Knowing which way

利用波长长的光源，散射时光子对电子扰动足够小，$p = \frac{h}{\lambda}$

但是波长长到狭缝间隔的尺度，我们将无法区分电子从哪个狭缝通过！

→ 分布(c) 干涉图像

区分电子从哪个狭缝通过，则无干涉 → 体现粒子性  
不区分电子从哪个狭缝通过，则有干涉 → 体现波动性

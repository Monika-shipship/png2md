# 一维定态问题-无限深方势阱

相应的波函数为
$$
u(x)=\begin{cases}
A\sin\frac{n\pi}{a}x, & 0 \leq x \leq a \\
0, & x < 0, or, x > a
\end{cases}
$$

(3) 波函数归一化
$$
\int_{-\infty}^{+\infty} u^*(x)u(x)dx = A^2\int_0^a \sin^2\frac{n\pi}{a}xdx = 1
$$
$$
\rightarrow A = \sqrt{\frac{2}{a}}
$$

## Figure & Layout Description

该PPT页面采用左右分栏布局，整体设计简洁明了，以黑白色为主，辅以蓝色强调元素。

页面顶部是黑色粗体标题"一维定态问题-无限深方势阱"，下方有一条细长的灰色水平线作为分隔。

左侧区域（约占页面2/3宽度）包含理论推导内容：
- 第一行是黑色普通字体的说明文字"相应的波函数为"
- 下方是分段函数表达式，用大括号表示：第一行是$A\sin\frac{n\pi}{a}x$，右侧标注条件"0 ≤ x ≤ a"；第二行是"0"，右侧标注条件"x < 0, or, x > a"
- 再下方是蓝色加粗标题"(3) 波函数归一化"，突出显示当前讨论的主题
- 接着是归一化条件的积分表达式：$\int_{-\infty}^{+\infty} u^*(x)u(x)dx = A^2\int_0^a \sin^2\frac{n\pi}{a}xdx = 1$
- 最底部有一个蓝色向右箭头，指向最终结果$A = \sqrt{\frac{2}{a}}$

右侧区域（约占页面1/3宽度）包含两个垂直排列的示意图：
- 上方是一个简化的矩形示意图，外框为黑色线条，内部标注字母"A"，图中包含两条平行的竖直黑线（代表势垒）和一个红色圆点（代表粒子位置）
- 下方是一个更详细的势阱图，具有坐标系：横轴标记为"x"，从"0"到"a"；纵轴标记为"E"
  - 左侧区域（x < 0）为灰色填充，标注"V(x)=∞ (barrier)"
  - 中间区域（0 ≤ x ≤ a）为白色，标注"V(x)=0 (well)"
  - 右侧区域（x > a）为灰色填充，标注"V(x)=∞ (barrier)"
  - 在x=0和x=a处有垂直黑线分隔区域
  - 在x=0附近有一个红色圆点表示粒子位置
  - 图形底部有蓝色文字标注"一维无限深方势阱"

整体视觉层次清晰，左侧为数学推导，右侧为物理图像，颜色使用有明确功能区分：黑色用于主要文字和公式，蓝色用于强调标题和关键结果，灰色用于势阱图中的势垒区域，红色用于突出粒子位置。

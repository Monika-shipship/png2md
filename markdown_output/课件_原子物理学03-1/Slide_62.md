# 一维定态问题-无限深方势阱

## 综合而言

$$
u(x) = 
\begin{cases} 
A e^{ikx} + B e^{-ikx}, & 0 \leq x \leq a \\
0, & x < 0, \text{or}, x > a 
\end{cases}
$$

$A, B$ 是待定系数。

## 按照波函数的标准条件

### (1) 在 $x=0$ 处连续

$$
u(x)\big|_{x=0} = A + B = 0 \implies A = -B
$$

则  
$u(x) = A(e^{ikx} - e^{-ikx})$  
$= A[\cos kx + i \sin kx - (\cos kx - i \sin kx)]$  
$= 2iA \sin kx \xrightarrow{2iA \to A} u(x) = A \sin kx$

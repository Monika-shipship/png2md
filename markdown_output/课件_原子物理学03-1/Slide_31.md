# 实物粒子的波动性–波粒二象性

## Knowing which way

调弱光源，电子与光子可能散射也可能不散射。

### 跟踪电子

- 电子被1看到，记录到 $P_1'$ → 分布(a)
- 电子被2看到，记录到 $P_2'$ → 分布(b)
- 电子没被看到，记录到 $P_{12}$ → 分布(c) 干涉图像

## Figure & Layout Description

页面顶部为黑色粗体主标题"实物粒子的波动性–波粒二象性"，标题下方有一条深灰色水平分割线。分割线下方中央位置有蓝色英文副标题"Knowing which way"。  
主体内容分为左右两部分：  
- **左侧**：黄色矩形光源图标向右发射三条箭头，指向带有两条蓝色竖线的双缝板（缝1在上，缝2在下）。缝1上方有紫色立方体及橙色向上箭头，缝2下方有紫色立方体及橙色向下箭头，双缝中间位置有红色星形散射点。  
- **右侧**：垂直排列三个概率分布图，从左到右依次标记为(a)(b)(c)：  
  - (a)为绿色平滑曲线，底部标注 $P_1'$；  
  - (b)为绿色平滑曲线，底部标注 $P_2'$；  
  - (c)为红色波浪状曲线，底部标注 $P_{12}$。
